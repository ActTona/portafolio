Buscador de Noticias

Objetivos:

Realizar un buscador de noticias especializado en un tipo especifico de contenido

Metodología:

Para realizar este proyecto se separo en 3 modulos principales, uno encargado a la busqueda de la información, el segundo optimización en la busqueda y el tercero la presentacion del contenido


• PRIMER MODULO

Se ocupó python con el uso de las librerias: pandas, sqlite3, scrapy, request, nltk, sklearn, os, time; el desarrollo consistio en realizar 2 tipos de busqueda, uno con busqueda RSS de una lista de paginas (RSS es una estructura que recopilar información en algunos sitios web) y el segundo extraer información de las noticias de google con una busqueda de palabras clave.

Ya con la información, se limpió y guardó en una base de datos, además se le asignó una calificación por palabras y por grupos de palabras de interés


• SEGUNDO MODULO

Con la información guardada, se prosiguió de una limpieza manual, para determinar que noticias eran las que nos interesaban, marcandolas con un 1 y cuales la que no, marcandolas con un 0, esto para poder hacer un análisis mas profundo.

Para la optimización se realizó una prueba diagnostica, con el fin de determinar la eficiencia de la busqueda de las noticias y tambien para determinar las modificaciones de como calificar el contenido, ademas de buscar la calificación optima de nuestras noticias, donde se optienen el menor numero de falsos negativos y falsos positivos (Entiendase como falso positivo las noticias marcadas como 1 por debajo de una calificacion de corte y como falso negativo las noticias marcadas con 0 que su calificacion este por arriba de la calificacion de corte)

Con los resultados optenidos se puntuaba diferente las noticias, para la cual se implementaron metodos con python para recalcular las calificaciones y actualizarlas en la base de datos.


• TERCER MODULO

Presentación de la información, se ocupó python con el uso las paqueterías flask y sqlite3 para levantar el servicio y hacer las consultas de las noticias, por el lado de la visualizacion de ocupo HTML,CSS y JavaScript con jinja

Resultados:

Dentro del análisis estadísto hecho, se llego a un area bajo la curva ROC de 0.85 lo cual es bastante bueno, además de que la información que se muestra se hace en la calificación de corte óptima

Automatizacion de reportes

Objetivos:

Disminuir el tiempo en la realización de algunos reportes

Metodología:

Se presentan 3 reportes en los que se automatizo el proceso


• PRIMER REPORTE

Este reporte se desarrollo en python con el paquete docx, se tiene un archivo excel que es generado por otro programa, del cual se contiene la infomacion necesaria para colocar en el documento

El proyecto consistio en pasar esa información a un reporte con un formato especifico en un archivo word, lo tardado o laborioso fue dar el formato de la salida


• SEGUNDO REPORTE

Este reporte se desarrollo en python con el paquete pptx, la finalidad generar una presentacion en automatico con datos calculados previamente, el diseño fue basado con la información anual del 2021, la idea era conectar este programa al progrma que calcula los resultados, para generar la presentacion del los resultados del año 2022


• TERCER REPORTE

Este reporte se desarrollo en excel con macros (VBA) y python (antes de que se implementara python en excel de forma nativa), la macro realiza peticiones al servidor de la base de datos local y despues agrega la información en el reporte, hay campos en el reporte que requieren hacer un calculo especifico y se tiene una base con un gran volumen de datos, la cual no puede procesar facilmente excel, por ello se hace una conexion a python para ejecutar los calculos necesarios y agilizar el proceso.

Se creo un pequeño formulario en VBA, para facilitar la interacción con el usuario. En el formulario se selecciona de una lista el año del que se quiere hacer el reporte y la base que se ocupara.

Resultados:

El primer reporte nos presento una ventaja de ahorro de hasta 2 hrs por reporte

El segundo reporte no se logro implementar por cambios en el los diseños, pero se puede ver el resuldo de lo que generaba en las imagenes de abajo

El tercer reporte nos ahorro un tiempo de hasta 3 semanas de trababajo
